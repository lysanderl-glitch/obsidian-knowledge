##### 3. Convert HTML & Build Request (核心)
- **类型**: Code
- **模式**: runOnceForAllItems

**关键代码逻辑**：
```javascript
const html = $('Fetch Article from Website').first().json.data || '';
const accessToken = $('Get WeChat Access Token').first().json.access_token;

// 1. 提取标题
const titleMatch = html.match(/<h1[^>]*>([\s\S]*?)<\/h1>/i);
const title = titleMatch ? titleMatch[1].replace(/<[^>]+>/g, '').trim() : 'Claude Code + 微信接入';

// 2. 定位文章内容边界（关键！）
const proseClassIndex = html.indexOf('class="prose', articleStart);
const proseOpenTagEnd = html.indexOf('>', proseClassIndex) + 1;
const footerStartInArticle = html.indexOf('<footer', proseClassIndex);

// 3. 提取文章内容
const proseContent = html.substring(proseOpenTagEnd, footerStartInArticle);

// 4. HTML 清洗：移除 class, style, nav, script, aside
let articleHtml = proseContent
  .replace(/<nav[\s\S]*?<\/nav>/gi, '')
  .replace(/<script[\s\S]*?<\/script>/gi, '')
  .replace(/<style[\s\S]*?<\/style>/gi, '')
  .replace(/<aside[\s\S]*?<\/aside>/gi, '')
  .replace(/\s*class="[^"]*"/gi, '')
  .replace(/\s*style="[^"]*"/gi, '');

// 5. 转换为微信支持的 HTML（内联样式）
articleHtml = articleHtml
  .replace(/<code>/gi, '<code style="background:#f4f4f4;padding:2px 6px;border-radius:3px;">')
  .replace(/<pre>/gi, '<pre style="background:#f6f8fa;padding:16px;overflow-x:auto;border-radius:6px;">')
  .replace(/<h2[^>]*>/gi, '<p style="font-size:20px;font-weight:bold;margin:28px 0 14px;">')
  .replace(/<\/h2>/gi, '</p>')
  .replace(/<p([^>]*)>/gi, (m, attrs) => {
    if (attrs.includes('style=')) return m;
    return '<p style="margin:14px 0;line-height:1.9;">';
  })
  // ... 更多转换规则
  ;

// 6. 构建请求体
const bodyObj = {
  articles: [{
    title: title,
    author: 'Lysander',
    digest: '摘要说明',
    content: wechatHtml,
    content_source_url: 'https://lysander.bond/blog/...',
    thumb_media_id: thumbMediaId,
```

## 四、关键问题与解决方案

### 4.1 微信公众号 HTML 限制

**问题**: 微信公众号只支持有限的 HTML 子集，不支持 `class`（Tailwind 样式）。

**解决方案**: 
1. 移除所有 `class` 和 `style` 属性
2. 使用内联样式替代（如 `<p style="margin:14px 0;">`）
3. 转换表格、列表等标签为微信支持的格式

### 4.2 文章内容提取边界问题

**问题**: 正则匹配容易找到错误的嵌套标签边界，导致内容截断。

**解决方案**: 使用精确的位置计算
```javascript
// 找到 prose 类的开始位置
const proseClassIndex = html.indexOf('class="prose', articleStart);
const proseOpenTagEnd = html.indexOf('>', proseClassIndex) + 1;
// 找到 footer 的开始位置（在 article 内部）
const footerStartInArticle = html.indexOf('<footer', proseClassIndex);
// 提取两者之间的内容
const proseContent = html.substring(proseOpenTagEnd, footerStartInArticle);
```

### 4.3 access_token 传递问题

**问题**: n8n HTTP Request 节点中 `{{ }}` 表达式处理有问题。

**解决方案**: 使用 `sendQuery: true` + `queryParameters` 方式传递
```javascript
// 错误方式：在 URL 中使用表达式
url: "https://api.weixin.qq.com/cgi-bin/draft/add?access_token={{ $json.access_token }}"

// 正确方式：使用 queryParameters
sendQuery: true,
queryParameters: {
  parameters: [{
    name: "access_token",
    value: "={{ $json.access_token }}"
  }]
}
```

### 4.4 HTTP Request body 类型问题

**问题**: `jsonBody` 接收对象，但 Code 节点返回的是字符串。

**解决方案**: 使用 `specifyBody: "string"` + `rawContentType: "application/json"`
```javascript
sendBody: true,
specifyBody: "string",
contentType: "raw",
rawContentType: "application/json",
body: "={{ $json.body }}"
```

## 五、完整操作步骤

### 5.1 发布博客到微信公众号

1. 打开 https://n8n.lysander.bond/workflow/LGkeWFUdYx5X7vgP
2. 修改 `Fetch Article from Website` 节点的 URL 为目标文章地址
3. 修改 `Convert HTML & Build Request` 中的 `content_source_url` 和 `thumb_media_id`
4. 点击 "Execute Workflow" 手动触发
5. 登录微信公众平台，在「草稿箱」查看发布的文章

### 5.2 新增文章流程

1. 在 `website/src/pages/blog/` 创建新的 `.astro` 文件
2. 编写文章内容（使用 Tailwind class 样式）
3. 运行 `npm run build` 构建
4. 在 n8n 工作流中更新 URL
5. 执行工作流发布到微信公众号草稿箱
6. 在微信公众平台编辑并发布
