# n8n 工作流开发经验

## 核心规则

### 节点执行策略

- 使用 `executeOnce: true` 让节点只执行一次（当接收多个 items 时）
- 使用 `.to()` 链式连接节点
- 使用 `$('NodeName').item.json.field` 引用上游节点输出
- 使用 `expr('{{ ... }}')` 在参数中使用表达式

### 执行流程理解

当 "Get All Users" 返回 N 个用户时，下游节点会执行 N 次，除非设置了 `executeOnce`。
