# 技术负责人 - 招聘 JD

## 关于我们

我们是数字化建筑运营管理公司，正在构建基于AI的智能团队管理系统。我们的愿景是打造"第二大脑"，让AI成为真正的数字同事。

**核心技术栈：**
- Claude Code / AI Agent
- CrewAI 多Agent框架
- Python / Go
- n8n 工作流自动化
- Obsidian 知识管理
- IoT 数据平台

## 岗位职责

### 1. 技术战略与架构
- 制定研发团队技术发展方向
- 设计AI驱动系统的架构方案
- 决策技术选型（保持AI-first思维）

### 2. AI系统工程
- 负责AI Agent系统的设计、集成与优化
- 推动AI-first开发流程落地
- 建立AI辅助编程规范（GitHub Copilot/Claude Code）

### 3. 团队技术管理
- 代码评审与质量控制
- 技术难题攻关
- 指导团队成员AI能力提升
- 建立SRE文化

### 4. 研发效能
- 设计高效研发流程
- 推动自动化测试与CI/CD
- 优化开发体验

## 任职要求

### 必须具备
- 5年以上软件开发经验
- 精通至少一门后端语言（Python/Go/Java）
- 有大型系统架构设计经验
- 理解AI/LLM基本原理，有AI产品开发经验
- 具备英文读写能力（阅读英文文档）

### 加分项
- 有多Agent系统开发经验
- 有CrewAI/LangChain使用经验
- 有IoT/数字化建筑相关经验
- 有Startup经验
- GitHub Contributor
